# SpaceX GraphQL Gateway

A GraphQL gateway for [r-spacex/SpaceX-API](https://github.com/r-spacex/SpaceX-API)

### Installation

```bash
$ yarn
```

### Running the app

```bash
# development mode
$ yarn start

# production mode
$ yarn build && yarn serve
```

### Test

```bash
# static analysis
$ yarn static:analysis

# unit test
$ yarn test:unit
```
